# Herramientas

- `install_core_from_github.py`: clona M5GP 2.0 y coloca los módulos en la única carpeta `core/src/m5gp`.
- `verify_structure.py`: verifica que no exista la duplicidad `core/methods/src`.

## Publicación en GitHub

- `check_github_repository.py`: comprueba que secretos, datos de ejecución y fuentes originales de M5GP 2.0 no estén bajo control de Git.
- `publish_to_github.sh`: inicializa el repositorio, confirma los archivos permitidos y publica en `armandocardenasf/GenLab-m5gp`.
